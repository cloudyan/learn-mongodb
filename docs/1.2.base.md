# 简单使用

- 简单信息查询
- 简单增删改查

```bash
db.version()
show dbs

use test;

# 默认 test
db
db.getName()

db.stats()

> db.getMongo()
connection to 127.0.0.1:27017
```

## 增 insert

```bash
$ mongo
> db.nblog.insert({name: 'jack', age: 20})
> db.nblog.find()
```

## 删 remove

不带参数将删除所有数据，是很危险的操作，这在mongodb中是一个不可撤回的操作，三思而后行

```bash
> db.nblog.remove({name: 'test'})
```

## 改 update

```bash
> db.nblog.update({name: 'test'}, {name: 'test'}, age: 12)
```

## 查 find

```bash
> db.nblog.find()
> db.nblog.find({name: 'test'})
> db.nblog.count()
```
