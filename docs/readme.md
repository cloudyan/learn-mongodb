## 场景分析

针对不同场景选择使用不同的解决方案，一旦选择错误，代价是巨大的。

使用 MongoDB，务必要深刻了解其适用场景，请阅读这个[案例](http://www.sarahmei.com/blog/2013/11/11/why-you-should-never-use-mongodb/)，让你在选择时更谨慎一些。
